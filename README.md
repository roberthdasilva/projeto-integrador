# Projeto Integrador (Equipe Desenvolvimento)
 
Este documento descreve o passo a passo para configurar e iniciar a aplicação Web (HTML, CSS e JavaScript) disponível no repositório: [projeto-integrador](https://github.com/roberthdasilva/projeto-integrador/).

## Pré-requisitos
- Editor de código. (VSCode, IntelliJ...)
- Extensão Live Server. (opcional)
- Navegador moderno. (Google Chrome, FireFox, Opera, etc)

## Passo a Passo

1. **Clonar o repositório**
   Clone o repositório do projeto.
    ```bash
   git clone https://github.com/roberthdasilva/projeto-integrador.git
   cd projeto-integrador
   ```
    1.1 **Atualizar a main**
        Atualize a branch main.
        ```bash
        git pull
        ```
   
2. **Troque para a develop**
   Troque para a branch de desenvolvimento, e atualize ela.
    ```bash
    git checkout develop
    git pull
   ```

3. **Crie sua branch**
   Crie sua branch e faça as alterações.
    ```bash
    git checkout -b nome-da-sua-branch
   ```
